Listas OL e UL

O Poder das Listas

Nós usamos listas a todo momento, seja para fazer compras, uma lista de tarefas ou uma receita de bolo e na
HTML não é diferente. Então vamos aprender um pouco sobre elas.

OL

Orderd Lists- Aquelas listas onde a ordem dos itens é algo muito importante, como aquela lista de carro dos sonhos.

  1. BMW
  2. Maserati
  3. Lamborghini
  4. Porsche
  5. Bugatti
  6. Ferrari

UL

Se você compreendeu a criação de listas ordenadas, com certeza vai entender as unordered lists, também
chamadas de listas com marcadores, que são aquelas onde a ordem dos itens não influenciará no significado da lista.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Misturando as Coisas

Podemos criar listas mistas, configurando listas dentro de outras listas. Veja o exemplo:

  1. Item 1
    1. Item 1.1
    2. Item 1.2
  2. Item 2
  3. Item 3
  4. Item 4
  5. item 5
